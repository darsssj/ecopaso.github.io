## 🚀 Cómo subir a GitHub Pages (paso a paso)

### 1. Crear el repositorio
1. Ve a [github.com](https://github.com) e inicia sesión
2. Clic en **"New repository"** (botón verde)
3. Nombre del repositorio: `ecopaso` (o el que prefieras)
4. Déjalo en **Public**
5. **NO** marques "Add a README file"
6. Clic en **"Create repository"**

### 2. Subir los archivos
1. En la página del repositorio recién creado, clic en **"uploading an existing file"**
2. Arrastra **TODOS** los archivos y la carpeta `img/` al área de carga
3. En el mensaje de commit escribe: `Sitio web ECOPASO`
4. Clic en **"Commit changes"**

### 3. Activar GitHub Pages
1. Ve a **Settings** (pestaña del repositorio)
2. En el menú izquierdo clic en **"Pages"**
3. En **"Branch"** selecciona `main` y carpeta `/ (root)`
4. Clic en **"Save"**
5. Espera 1–2 minutos

### 4. ¡Listo! 🎉
Tu página estará disponible en:
```
https://TU_USUARIO.github.io/ecopaso/
```
